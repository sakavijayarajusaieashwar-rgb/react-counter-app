# React Counter App

A simple counter application built using React.

## Features

* Increase count
* Decrease count
* Dynamic color updates
* Responsive UI
* React state management using `useState`

## Tech Stack

* React
* JavaScript
* CSS
* Vite

## Concepts Learned

* React Components
* useState Hook
* Event Handling
* Conditional Rendering
* Dynamic Styling

## Project Setup

Clone the repository:

```bash
git clone https://github.com/sakavijayarajusaieashwar/react-counter-app.git
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
